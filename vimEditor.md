# Commands for Vim Editor:
- vi -> open vim editor.
    - ex: vi file.txt
- i -> enter into insert mode.
    - ex: i
- esc -> edit mode.
    - ex: esc
- ZZ -> save & quit.
    - ex: esc then press ZZ
- :q! -> exit without save.
    - ex: esc then press :q!
- :w -> save without exit.
    -ex: esc then :w
- :wq -> save & exit
    - ex: esc then :wq
- delete entire line -> dd
- delete single char -> x
- undo -> u
- :set number
- move up by press -> k
- move down by press -> j
- move left by press -> h
- move right by press -> l
